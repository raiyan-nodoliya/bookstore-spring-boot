# BookStore Management System (Spring Boot + Thymeleaf + JPA)

## Features
- Add Book (Create)
- View All Books (Read)
- Edit Book (Update)
- Delete Book (Delete)
- My Books List (optional)

## Tech Stack
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA (Hibernate)
- Thymeleaf
- H2 Database (In-memory)
- Bootstrap 5

## How to Run
1. Clone the repo
   ```bash
   git clone <repo-url>
   cd <project-folder>

2.Run the application
  mvn spring-boot:run
  
3.Open in browser
  Home: http://localhost:8080/
  Available Books: http://localhost:8080/available_book
  Register Book: http://localhost:8080/book_register
  H2 Console (Optional)
  URL: http://localhost:8080/h2-console
  JDBC URL: jdbc:h2:mem:testdb
  Username: sa
  Password: (blank)
